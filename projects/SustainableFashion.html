Project Lead: Ashley Philip

Past Contributions: Jane Nguyen, Celina Sarona, Joan Suh, Jennifer Lee

Motivation:

Create a website that allows people to trade clothes with others nearby, preventing high clothing consumption while still allowing people to revamp their wardrobe. The target audience is college students because they are frequently buying clothes and can also trade with other students on campus safely.

Methods:

We created the website using React.js, Javascript, HTML, and CSS for the framework. We also created a Figma mockup of our application so we were able to have a clear vision. Then, we implemented users and posting features using Google Firebase and added some additional styling using Bootstrap. The website is still being polished but it has made a lot of progress and the basic features are working!
